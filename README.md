# 🌱 Smart Agriculture AI Assistant

## 📌 Overview
Smart Agriculture AI Assistant is an AI-powered application designed to help farmers identify crop diseases and receive agriculture-related guidance. The project combines machine learning, natural language processing, and an interactive Streamlit interface.

## ✨ Features
- 🌿 Crop disease prediction using deep learning
- 🤖 AI-powered agriculture question-answering assistant
- 📊 Data visualization and performance analysis
- 💻 User-friendly Streamlit web application

## 🛠️ Technologies Used
- Python
- TensorFlow
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📂 Project Files
- `app.py` – Main Streamlit application
- `train_disease.py` – Model training
- `predict_disease.py` – Disease prediction
- `graphs.py` – Performance graphs and visualization

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/SomyaHedau003/Smart-Agriculture-AI-Assistant.git
```

2. Install the required libraries:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run app.py
```

## 📈 Results
The model predicts crop diseases and provides agriculture-related recommendations through an interactive web interface.
